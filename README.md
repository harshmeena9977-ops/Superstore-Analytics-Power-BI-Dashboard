# Superstore Analytics Dashboard – Power BI Project

## 📌 Overview
End-to-end Power BI dashboard project built on the Superstore dataset.  
Covers data cleaning, modeling, DAX measures, and interactive visuals with a collapsible slicer panel.

## 🎯 Objectives
- Track Sales, Profit, and Returns across regions and categories
- Identify top-performing and loss-making products
- Enable dynamic filtering with slicer panel (Select Order, Top N Orders)
- Provide recruiter-ready portfolio artifact

## ⚙️ Steps Covered
1. Data Transformation – Power Query cleaning and shaping  
2. Data Modeling – Relationships, Date table setup  
3. DAX Measures – Sales, Profit, YoY Growth, % Returns  
4. Visuals – KPIs, Line Chart, Bar Chart, Map, Donut Chart  
5. Slicer Panel – Collapsible panel using Bookmarks + Selection Pane  

## 📊 Key Insights
- California generated the highest profit  
- Tables category showed consistent losses  
- Technology Copiers were the most profitable product line  
- Consumer segment contributed the most to sales
